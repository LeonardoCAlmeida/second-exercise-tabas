# README

* Ruby version
  - 2.6.5

* Rails version
  - 5.2.6

* System dependencies
  - Enviroment to run Ruby on Rails, Database
  
* Configuration
  - Fill database.yml with your datas.
  - Install all the gems. (bundle install)

* Database creation
  - Used PostgreSQL

* Database initialization
  - Run in your terminal
      $ rake db:create 
      $ rake db:migrate
      $ rake db:seed
